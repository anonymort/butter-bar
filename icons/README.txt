Butter Bar logo package
- butter-bar-logo.svg: master vector source (1024 canvas)
- butter-bar-logo-1024.png: master raster export
- butter-bar-logo@1x.png / @2x.png / @3x.png: convenience raster exports
- ButterBar.iconset/: macOS app icon PNG set
- ButterBar.icns: macOS icon container

Design notes:
- Mark is kept well inside the safe area for Big Sur+ squircle masking.
- Core metaphor: a butter pat resting on a bar/shelf, with a carved play symbol.
- Contrast and silhouette were kept simple so the icon remains legible at 16 px.
